# Contributing to GSF Blueprint

GSF is an open-collaboration system blueprint.
All contributions are design, discussion, and research oriented.

## What’s Welcome
- Architecture critiques
- Security improvements
- Governance / legal framing suggestions
- Diagrams, flowcharts, threat models
- Proof-of-concept designs (optional)

## What’s Not Allowed
- Claiming this is a finished product
- Promising ROI or profits
- Selling tokens or investments
- Spam, marketing, or broker promotion

## Contribution Format
- 1 Pull Request = 1 main topic
- Explain clearly:
  - What problem you found
  - Your proposed solution
  - The impact on the GSF blueprint

## Discussions
- Use Issues for concept debates  
- Use Pull Requests for document / design changes
